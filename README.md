# mocha-chai-practice
 
This is practice reporsitory.

## Setup Instructions

- npm install mocha --save-dev
- npm install chai --save-dev
- Update the `test` field in package.json to mocha
- Create a folder named `test` in root
- Write all your tests inside `test` folder

## How to run the test
- npm test
